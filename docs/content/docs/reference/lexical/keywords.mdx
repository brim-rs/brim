Keywords are reserved identifiers that have special meanings and cannot be used as names for variables, functions, or other identifiers. This page provides a list of all keywords in Brim, along with their descriptions.

## List of Keywords

- `type`
- `as`
- `fn`
- `let`
- `if`
- `else`
- `return`
- `true`
- `false`
- `null`
- `while`
- `for`
- `in`
- `break`
- `continue`
- `use`
- `pub`
- `from`
- `catch`
- `loop`
- `struct`
- `impl`
- `trait`
- `then`
- `const`
- `enum`
